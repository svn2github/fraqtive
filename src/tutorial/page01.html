Welcome to Fraqtive

Fraqtive is a Mandelbrot family fractal generator. It uses very fast algorithms supporting SSE2 and multi-core processors. It generates high quality anti-aliased images and renders 3D scenes using OpenGL. It allows real-time navigation and dynamic generation of the Julia fractal preview.

This tutorial will help you quickly start working with Fraqtive. You can still use the application while this window is open. You can open this tutorial later by selecting Help > Quick Tutorial.
